## CRM Admin Full Stack Application
## Technologies Used

Backend:
  
    Java 17
    
    Spring Boot
    
    MySQL
    
    Maven

Frontend:
  
    React JS
    
    Axios
    
    HTML
    
    CSS

## Backend Setup
  
    Go to CRM-backend folder
    
    Update application.properties with your MySQL username & password
    
    Run:
    
    mvn spring-boot:run
    
    http://localhost:8080

## Frontend Setup
  
    Go to CRM-frontend folder
    
    Run:
    
    npm install
    
    npm start
    
    http://localhost:3000

## Features
 
    Add Student
    
    View Student List
    
    View Student Details
    
    Update Student Information
    
    Delete Student (with confirmation popup)
    
    REST API Integration
    
    MySQL Database Connectivity
